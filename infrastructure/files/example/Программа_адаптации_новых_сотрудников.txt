ПРОГРАММА АДАПТАЦИИ НОВЫХ СОТРУДНИКОВ

1. ОБЩИЕ ПОЛОЖЕНИЯ
1.1. Цель программы: обеспечить плавное вхождение нового сотрудника в рабочий процесс, ознакомить с корпоративной культурой и снизить уровень стресса в первые месяцы работы.

1.2. Сроки адаптации:

Для рядовых сотрудников – 1 месяц

Для руководителей – 3 месяца

Для сотрудников на испытательном сроке – весь период испытательного срока

1.3. Участники процесса:

HR-менеджер

Наставник (из числа опытных сотрудников)

Непосредственный руководитель

Коллеги по отделу

2. ЭТАПЫ АДАПТАЦИИ
2.1. Подготовительный этап (до выхода сотрудника)
Оформление рабочего места (ПК, телефон, канцелярия)

Подготовка пакета документов:

Трудовая книжка

Личная карточка

Ознакомительные материалы

Назначение наставника (приказом по компании)

2.2. Первый рабочий день
Экскурсия по офису

Знакомство с коллективом

Вводный инструктаж по технике безопасности

Выдача корпоративных реквизитов (почта, пропуск)

2.3. Первая неделя
Обучение основам работы:

Внутренние регламенты

CRM-система

Корпоративный портал

Знакомство с должностной инструкцией

Первая встреча с руководителем

2.4. Первый месяц
Постепенное увеличение нагрузки

Промежуточная оценка результатов

Корректировка плана адаптации

3. РОЛИ И ОБЯЗАННОСТИ
3.1. HR-отдел:

Организация welcome-тренинга

Проведение анкетирования (на 1-й день, через 1 месяц)

Контроль общего процесса адаптации

3.2. Наставник:

Ежедневные консультации

Помощь в решении текущих задач

Обратная связь для руководителя

3.3. Руководитель:

Постановка задач

Оценка промежуточных результатов

Принятие решения о завершении адаптации

4. ИНСТРУМЕНТЫ АДАПТАЦИИ
4.1. Welcome-pack:

Оргсхема компании

Контакты ключевых сотрудников

Глоссарий корпоративных терминов

4.2. Чек-лист адаптации:

Получение доступа к системам

Прохождение обязательного обучения

Знакомство с клиентами/проектами

4.3. Обратная связь:

Анкета "Первые впечатления" (через 3 дня)

Опрос "Итоги месяца"

Личная беседа с HR

5. КРИТЕРИИ УСПЕШНОЙ АДАПТАЦИИ
5.1. Для сотрудника:

Уверенное выполнение базовых задач

Полная интеграция в коллектив

Понимание корпоративных ценностей

5.2. Для компании:

Сокращение текучести кадров

Снижение количества ошибок новичков

Повышение лояльности сотрудников

6. ПРИЛОЖЕНИЯ
Чек-лист для наставника

Шаблон плана адаптации

Форма обратной связи
